Quick Glance

Method Endpoint Description
POST /package Create a new package.
DELETE /package/byName/{name} Delete all versions of this package.
GET /package/byName/{name} Return the history of this package (all versions).
DELETE /package/{id} Delete this versions of the package.
GET /package/{id} Return this package.
PUT /package/{id} Update this version of the package.
GET /package/{id}/rate Get the rating for this package.
POST /packages Get packages.
DELETE /reset Reset registry.
POST /user Make a new user.
